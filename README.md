# Boilerplate for TypeScript, React & Storybook with Webpack 4

A boilerplate project for using TypeScript 3, React 16 and Storybook together with Webpack 4.

## Quickstart

With yarn

```bash
yarn
yarn start:storybook
```

With npm

```bash
npm install
npm run start:storybook
```
